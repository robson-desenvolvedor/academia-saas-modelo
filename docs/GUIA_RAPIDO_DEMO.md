# Guia rápido da demo

## 1. Entrar como Academia

Use `academia@demo.com / 123456` ou o botão de acesso rápido.

A academia consegue:

- ver o painel geral;
- acessar calendário completo;
- criar aulas, avaliações e horários;
- gerenciar agendamentos;
- cadastrar alunos;
- cadastrar profissionais;
- ativar e bloquear usuários;
- exportar CSV;
- visualizar relatórios.

## 2. Entrar como Personal

Use `personal@demo.com / 123456`.

O personal consegue:

- ver apenas a própria agenda;
- criar horários próprios;
- acompanhar seus alunos;
- adicionar disponibilidade;
- concluir ou cancelar sessões;
- editar perfil simulado.

## 3. Entrar como Aluno

Use `aluno@demo.com / 123456`.

O aluno consegue:

- ver aulas e horários disponíveis;
- reservar horários;
- cancelar as próprias reservas;
- acompanhar créditos;
- ver treino semanal simulado;
- consultar personais disponíveis.

## 4. Observação

A demo não usa banco de dados para facilitar publicação na Vercel. Os dados ficam no navegador por `localStorage`. O botão "Resetar demo" limpa e recarrega os dados iniciais.
